🔧 Technical Implementation

Responsive Picture Element:

<picture>
  <source srcset="images/responsive/regular3-2-1x.webp 1x, 
                  images/responsive/regular3-2-2x.webp 2x" 
          type="image/webp">
  <img src="images/responsive/regular3-2-1x.jpg" 
       srcset="images/responsive/regular3-2-1x.jpg 1x"
       alt="Tour image" 
       width="380" 
       height="380">
</picture>

Browser Selection Logic:

  • ✅ WebP supported + 1x display → 22KB WebP
  • ✅ WebP supported + 2x display → 71KB WebP
  • ✅ WebP not supported → 31KB JPEG fallback
  • ✅ All cases → Correct dimensions served
